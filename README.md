# Tugas Personal 2 - Programming for Business

Repositori ini berisi implementasi kode Java untuk Tugas Personal 2 (TP2) mata kuliah Programming for Business.

## Daftar File Program:
1. **VehicleSystem.java**: Implementasi Konsep Inheritance dan Encapsulation (Pewarisan dan Pengkapsulan).
2. **PaymentSystem.java**: Implementasi Konsep Polymorphism (Overloading dan Overriding) dalam skenario sistem pembayaran.
3. **EmployeeSystem.java**: Implementasi Konsep Abstract Class dalam sistem manajemen karyawan.
4. **ExceptionAndPatternSystem.java**: Implementasi Exception Handling (try-catch-finally) dan Pattern Matching menggunakan `instanceof`.
5. **AdvancedJavaSystem.java**: Implementasi fitur modern Java seperti Lambda Expressions, Sealed Classes, dan Final Class.

## Cara Menjalankan:
Pastikan Anda memiliki Java JDK 17 atau yang lebih baru terinstal. Anda dapat mengkompilasi dan menjalankan setiap file menggunakan perintah:
```bash
javac NamaFile.java
java NamaFile
```
